# Как работает Nova Architect

## Одной фразой

Open-source CLI, который ставишь через npm. Запускаешь в любом проекте — он поднимает приложение, открывает браузер с AI-оверлеем, и ты голосом/кликами говоришь что менять. AI меняет код, hot reload показывает результат.

---

## Установка — одна команда

```bash
npm install -g nova-architect
```

Всё. Не нужен .NET runtime, не нужен Electron, не нужен Docker. Один npm-пакет.

---

## Запуск — одна команда

```bash
cd ~/projects/my-app
nova
```

Система сама:
1. Находит package.json / .csproj / requirements.txt → понимает стек
2. Сканирует код → строит карту: страницы, компоненты, API, модели
3. Запускает dev-сервер (npm run dev, dotnet run, ...)
4. Поднимает прокси перед dev-сервером
5. Открывает браузер на прокси-порту — твоё приложение + тонкий AI-оверлей

---

## Как работает оверлей

Никакого Electron. Прокси-сервер инжектит маленький скрипт (~20KB) в каждую HTML-страницу. Этот скрипт рисует плавающую «пилюлю» в углу экрана: кнопка микрофона, индикатор статуса.

Работает в любом браузере — Chrome, Firefox, Safari. Используешь свои привычные dev tools.

---

## Пошагово: ты просишь — AI делает

### 1. Ты кликаешь или говоришь

Нажал на элемент в приложении → оверлей подсвечивает его, показывает поле ввода.
Пишешь: "добавь поиск сюда" или говоришь голосом.

### 2. Оверлей собирает контекст

- Скриншот текущей страницы (html2canvas)
- Координаты клика + HTML-снепшот выбранного элемента
- Твой текст / голосовая команда
- Ошибки из console.error (если есть)

Всё это отправляется через WebSocket в CLI.

### 3. AI анализирует

CLI отправляет в мультимодальную модель:
- Скриншот
- Твои слова
- Карту проекта (какие файлы, роуты, компоненты)
- Контекст нужных файлов

Модель отвечает: "нужно создать компонент SearchInput и вставить его в /customers page".

### 4. Классификатор выбирает полосу

| Что просят | Полоса | Время | Стоимость |
|-----------|--------|-------|-----------|
| Поменять цвет, текст, отступ | 1 — Instant | < 2 сек | Бесплатно (Ollama) |
| Добавить поле, компонент, эндпоинт | 2 — Fast | 10-30 сек | ~$0.001 |
| Новая фича (фронт + бэк + БД) | 3 — Standard | 1-5 мин | ~$0.05 |
| Рефакторинг, миграция | 4 — Background | Фоном | ~$0.50 |

### 5. Исполнитель меняет код

- Полоса 1: правка AST/regex, без обращения к API
- Полоса 2: один вызов быстрой модели → diff → применить
- Полоса 3: декомпозиция → несколько вызовов → несколько файлов
- Полоса 4: фоновый агент, работает автономно

Каждое изменение — коммит в ветку `nova/*`.

### 6. Hot reload — видишь результат

Dev-сервер подхватывает изменённый файл → страница обновляется → ты видишь результат прямо в браузере.

Оверлей показывает тост: "Добавил SearchInput на /customers".

## Режимы

```bash
nova             # Полный: браузер + оверлей + голос + всё
nova chat        # Только терминал, как Claude Code. Без браузера.
nova watch       # Пассивно наблюдает, копит предложения. Ничего не меняет.
nova tasks       # Показывает очередь задач
nova review      # Показывает все изменения для ревью
```

## Лицензия и цена

Код полностью открыт на GitHub — source-available (не MIT). Модель как у Remotion:

- **Бесплатно** — для индивидуалов и команд до 3 человек. Полный функционал, без ограничений. Навсегда.
- **Company License** — для компаний от 4 человек. $25/мес за разработчика. Если компания из 10 человек экономит по часу в день — $250/мес за это смешная цена.

AI-модели — отдельно, ты приносишь свой ключ:
- **OpenRouter** — доступ ко всем моделям, одним ключом
- **Anthropic** — Claude напрямую
- **OpenAI** — GPT модели
- **Ollama** — полностью бесплатно, локально, для простых правок

В будущем: **Nova Cloud** — managed-сервис, где не нужно настраивать ключи, всё работает из коробки.

## Конфиг — `nova.toml`

Один файл в корне проекта. Коммитится в репо — вся команда шарит настройки.

```toml
[project]
dev_command = "npm run dev"
port = 3000

[models]
fast = "openrouter/qwen-2.5-coder-7b"
strong = "anthropic/claude-sonnet-4"
local = true

[behavior]
auto_commit = false
branch_prefix = "nova/"
```

## Сравнение

| | Cursor / Claude Code | Bolt / Lovable | Nova Architect |
|--|---------------------|----------------|----------------|
| Работает с существующим репо | Да | Нет | Да |
| Интерфейс | IDE / терминал | Браузер (их платформа) | Браузер (твоё приложение) |
| Способ ввода | Текст | Текст + клик | Клик + голос + текст |
| Нужно открывать IDE | Да | Нет | Нет |
| Видишь результат сразу | Нет (preview) | Да | Да (hot reload) |
| Код открыт | Нет / частично | Нет | Да (source-available) |
| Стоимость | $20+/мес | $20+/мес | Бесплатно до 3 чел (+ свой API ключ) |
